Xbox One Treiber installation
	1. GeräteManager öffnen
	2. Geräteeigenschaften öffnen
	3. Trreiber aktualisieren --> Auf dem Computer anch Treibersoftware suchen.
	4. Ordner auswählen.



Falls es nicht funktioniert hat man die Möglichkeit bei
	https://www.catalog.update.microsoft.com/Home.aspx

nach dem entsprechenden Trieber zu suchen, das geht jedoch nur mit dem InternetExplorer
