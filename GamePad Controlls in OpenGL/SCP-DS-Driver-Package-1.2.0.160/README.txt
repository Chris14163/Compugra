Wichtiges zur Installation für Playstation 3 Controller Traiber:
	1.ScpDriver als Administrator starten.
	2. Bei Windows 7 oder niedriger Force Install aktivieren, bei 8 und höher ist das nicht notwendig.
	3. Erfolgreiche installation falls:
		SCP DS3 Service
		Bus Device
		Bus Driver
		DS3 USB Driver 
	in der Install Summery aufgelistet wird.
